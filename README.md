# Laravel Crypto Balance Module

## Описание тестового задания

Необходимо разработать модуль учета крипто-баланса пользователя, который позволяет корректно и безопасно:
- зачислять средства
- списывать средства (выводы, платежи, комиссии)

Важно учитывать специфику блокчейна, асинхронность и риски. Стек технологий: PHP + Laravel.

Задание максимально простое, выполняется за 30-60 минут.

# CryptoBalanceMonitor

## 1. Обзор проекта
CryptoBalanceMonitor - Laravel-based API приложение для отслеживания балансов криптовалютных кошельков с течением времени.

## 2. Поддерживаемые валюты
- Bitcoin (BTC)
- Litecoin (LTC)
- Ethereum (ETH)

## 3. Архитектура
- REST API на Laravel
- CQRS-подход для команд и запросов
- Асинхронное обновление балансов через очереди
- Провайдеры балансов абстрагированы через интерфейсы

## 4. Провайдеры балансов
Поддерживаемые провайдеры:
- Blockchair (BTC, LTC)
- Etherscan (ETH)

По умолчанию включен FakeBalanceProvider для работы без внешних API.

## 5. Ограничения внешних API
Blockchair и Etherscan требуют API ключи и имеют строгие лимиты запросов. Для разработки и тестирования по умолчанию используется fake-провайдер.

## 6. Очереди и cron
Обновление балансов выполняется асинхронно:
- Cron запускается каждую минуту
- Диспатчит задачи в очередь
- Задачи получают балансы и сохраняют историю

## 7. Тестирование
Включает:
- Unit-тесты для сервисов и провайдеров
- Job-тесты для асинхронных обновлений
- Feature-тесты для API endpoints

Запуск тестов:
```shell
php artisan test
```
